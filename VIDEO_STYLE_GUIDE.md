# MyPropOps Video Brand Style Guide

## Overview
This guide ensures all MyPropOps video content maintains a consistent, professional look that positions us as the #1 Property Management Software.

---

## 1. Visual Identity

### The Device Frame
- **Always use**: Clean iPhone 16 Pro or minimal browser window mockup
- **Never show**: Raw, full-screen recordings with browser chrome
- **Aspect Ratio**: 9:16 (vertical) for social, 16:9 (horizontal) for website

### Background Canvas
| Use Case | Color | Hex Code |
|----------|-------|----------|
| Primary Background | Soft Slate | `#F4F7F9` |
| Alternative Background | Deep Navy | `#1E293B` |
| Accent Gradient | Blue to Indigo | `#2563EB → #4F46E5` |

### Logo Placement
- Position: **Top-right corner**
- Opacity: **40%**
- Size: **80px width max**
- Always visible throughout clip

---

## 2. Typography

### Primary Font
- **Headlines**: Inter Bold (or Manrope Bold)
- **Body Text**: Inter Medium
- **Accent Text**: Inter SemiBold

### Text Styling
| Element | Style | Example |
|---------|-------|---------|
| Hero Text | ALL CAPS, 48px | "STOP THE SPREADSHEET MADNESS" |
| Subtext | Sentence case, 24px | "Rent on autopilot" |
| CTA | ALL CAPS, Bold, 20px | "TRY FREE TODAY" |

### Text Placement
- **Lower Third**: Primary text overlays
- **Center**: Impact statements
- **Highlight Box**: White text on brand blue (`#2563EB`)

---

## 3. Motion & Transitions

### Transition Style: "Digital Snap"
- **Primary**: Directional swipes (left-to-right, simulating phone gestures)
- **Secondary**: Scale pops (element grows 110% then settles to 100%)
- **Never use**: Slow fades, spinning, or bouncy effects

### Pacing
- **Cuts per 6-second clip**: 3-4 cuts
- **Hold time per screen**: 1.5-2 seconds max
- **Tempo**: Fast, confident, decisive

### The Signature Animations
1. **Screen Entry**: Slide in from right with subtle shadow
2. **Feature Highlight**: Pulse glow around element
3. **Success State**: Green checkmark with particle burst
4. **Logo Resolve**: Fade in with subtle scale (1 second)

---

## 4. Color Psychology

### Action Colors
| Purpose | Color | Hex | Usage |
|---------|-------|-----|-------|
| Success/Money | Emerald | `#10B981` | Rent paid, task complete |
| Primary Action | Brand Blue | `#2563EB` | Buttons, CTAs |
| Alert/Urgent | Amber | `#F59E0B` | Deadlines, warnings |
| Highlight | Violet | `#8B5CF6` | New features |

### Neutrals
| Purpose | Color | Hex |
|---------|-------|-----|
| Dashboard BG | White | `#FFFFFF` |
| Card BG | Soft Gray | `#F8FAFC` |
| Text Primary | Slate | `#1E293B` |
| Text Secondary | Gray | `#64748B` |

---

## 5. Audio Branding

### Sound Elements
| Element | Description | When to Use |
|---------|-------------|-------------|
| **The Ping** | Glassy, satisfying notification | Success moments (rent paid, task done) |
| **The Whoosh** | Subtle air movement | Screen transitions |
| **The Click** | Soft button press | User interactions |
| **The Resolve** | Warm, confident tone | Logo appearance |

### Background Music
- **Style**: Lo-fi Corporate / Ambient Tech
- **BPM**: 110-120 BPM (energetic but not frantic)
- **Volume**: 40% under voiceover, 70% for B-roll

### Voice-Over Guidelines
- **Tone**: Confident, friendly, direct
- **Pace**: Quick but clear (not rushed)
- **Accent**: Neutral American English

## 7. Ending Sequence (All Videos)

**Duration**: 1 second

**Visual**:
1. Screen fades to white
2. MyPropOps logo fades in (centered)
3. URL appears below: "mypropops.com"
4. Tagline: "#1 Property Management Software"

**Audio**: The Resolve sound (warm, confident tone)

---

## 8. Production Checklist

Before publishing any video, verify:

- [ ] Logo in top-right at 40% opacity
- [ ] Background color matches brand palette
- [ ] Text uses Inter/Manrope font family
- [ ] Transitions are "Digital Snap" style
- [ ] Success moments use Emerald green
- [ ] Audio includes signature ping sounds
- [ ] Ends with 1-second logo resolve
- [ ] Total length is exactly 6 seconds (or specified duration)
- [ ] Exported at 1080p minimum (4K preferred)
